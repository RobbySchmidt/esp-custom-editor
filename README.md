# esp-custom-editor

Ein Nuxt 4 Projekt mit TailwindCSS v4.

## Setup

Abhängigkeiten installieren:

```bash
yarn install
```

## Development

Dev-Server auf `http://localhost:3000` starten:

```bash
yarn dev
```

## Production

Build für die Produktion erstellen:

```bash
yarn build
```

Produktions-Build lokal testen:

```bash
yarn preview
```

Mehr Infos in der [Nuxt-Dokumentation](https://nuxt.com/docs/getting-started/introduction).
